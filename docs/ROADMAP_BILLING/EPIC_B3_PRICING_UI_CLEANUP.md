- левую панель «Параметры расчёта» (`:78-101`),
- правую ониксовую панель «ЭКОНОМИКА ТАРИФА» (`:102-131`).

**Оставить** только карточку модели + одну CTA. Чтобы блок не выглядел пустым
(аудит §3 «адаптировать вёрстку»), заменить на компактную premium-карточку:

### 2. Модель «Фикс + %» — 3 режима, цена ÷2, 1.5%, период

**Удалить** (`PlansTab.tsx:137-190`):
- «Параметры расчёта» со слайдером оборота (`:140-160`),
- ониксовую «ЭКОНОМИКА КОМБО-ТАРИФА» (`:161-187`).

**Оставить/добавить** 3 режима фикс-части с ценой **÷2** от подписки и **периодом**:

| Режим | Было (фикс/мес) | Стало (÷2) | % |
|---|---|---|---|
| Старт | 990 | **495** | 1.5% |
| Про | 1490 → берём из каталога `pro/2` | **1245** | 1.5% |
| Бизнес | 1990 → `business/2` | **2995** | 1.5% |

> Цифры фикса — из бэкенда (`COMBO_FIXED` в `plans.py`, B2), не хардкодить в TSX.
> Тянем через каталог: `plans[id].monthly / 2`. Процент 1.5% — константа `t('combo.rate')`.

Добавить **селектор периода** (переиспользовать существующий блок «Период оплаты»
`:194-222` — он уже есть для подписки, вынести в под-компонент `PeriodSelector` и
показывать также в режиме `combo`). Период влияет **только на фикс** (аудит §3):
скидка применяется к фикс-части, процент 1.5% от периода не зависит.

Итоговый мини-расчёт (без «экономики»): просто строка
`t('combo.summary', { fixed: formatMoney(discountedFixed, currency), rate: 1.5 })` —
«Фикс {fixed}/мес + 1.5% с онлайн-оплат». CTA `Активировать Комбо-тариф` →
`activateModel({ mode:'combo', plan, period_months })`.

### 3. Блок «Ваша экономия» — точный расчёт + копирайтинг

Файл: `front/src/pages/dashboard/Billing/components/ui/SavingsIllustration.tsx`
(вызывается `PlansTab.tsx:277`). Формула уже корректна
(`monthly*period − monthly*period*(1−discount)`), задача — **копирайтинг + пустой стейт**:

- Точная экономия деньгами: `saved = monthly*period*discount` — показываем `formatMoney(saved, currency)`.
- Экономия «в месяцах» (аудит §3 «расчёт экономии месяцев вперёд»):
  `freeMonths = Math.floor(saved / discountedMonthly)` → `t('savings.freeMonths', { count })`
  «Это как N мес. бесплатно». **Только если `count >= 1`** — иначе не показываем (без фейка).
- Копирайтинг из i18n (`savings.*` в B0), не хардкод.
- Пустой стейт: `period === 1` (скидки нет) → блок показывает нейтральный
  «Выберите период от 6 мес. — и начнёте экономить», без нулей и тире.
- «Месяцев с нами» (если блок присутствует): `plan.status === 'none'` →
  `t('empty.monthsWithUs')` = «Вы ещё не купили подписку».

### 4. Общее

- Все `₽{x.toLocaleString('ru-RU')}` в `PlansTab`/`SavingsIllustration` → `formatMoney(x, currency)` (B0).
- Тексты моделей (`:60-62`) → `t('mode.*')`.
- Кнопки-модели (`:64`) остаются кастомными карточками-переключателями (это не «кнопка»
  в смысле кита, а radio-tile) — допустимо, но CTA-кнопки активации — строго `Button` кита.
