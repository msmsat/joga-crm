### ✅ 1.3 Убрать дерганье правой панели · 🟡 · 1:00
**Простыми словами:** при клике по карточке панель мигает и «прыгает». Причины в коде:
(а) при переключении клиента `ClientPanel` полностью перемонтируется — `key={client.id}`
(`ClientProfileSlider.tsx:898`); (б) профиль жёстко сбрасывается в `null` до ответа сервера
(`Clients.tsx:57`) — контент исчезает и появляется; (в) открытие/закрытие анимирует `width`
0↔420px, и сетка карточек `auto-fill minmax(320px)` пересчитывает колонки на каждом кадре.
**Что конкретно сделать:**
1. Убрать `key={client.id}`; локальные стейты панели (вкладка, открытые подпанели)
   сбрасывать эффектом по `client.id`.
2. Не сбрасывать данные в `null`: базовые поля уже есть в строке списка, профиль
   дозагружается тихо (после 2.1 — из кэша `client(id)`, повторное открытие мгновенно).
3. Открытие/закрытие: ширину панели не анимировать через `width` на каждый клик —
   при переключении клиентов панель уже открыта и ширина не должна меняться вовсе;
   для open/close оставить одну аккуратную анимацию (например, `transform` у панели +
   транзишен ширины обёртки только на open/close, не на смену клиента).
**Готово, когда:** переключение клиентов при открытой панели — без миганий и скачков сетки.
